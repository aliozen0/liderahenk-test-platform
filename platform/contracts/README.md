# Platform Contracts

This directory is the canonical home for declarative platform contracts.

- YAML and Markdown files here define runtime readiness, topology, registration,
  baseline, directory, and scenario-pack expectations.
- `platform_runtime/` and the validation scripts read from this directory.
- This directory is not a pytest test suite.

If you need contract tests, use `/home/huma/liderahenk-test/tests/contracts/`.
