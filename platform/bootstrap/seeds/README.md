# Bootstrap Seeds

Bu klasor, platformun urun yerine sahip oldugu baslangic verilerini dokumante eder.

Buradaki tohum veriler su alanlarla sinirlidir:

- LDAP kokleri
- varsayilan admin kullanicisi
- user/group/role OU agaci
- ajan bootstrap identity'leri
- acceptance ve baseline capture metadata'si

Urun davranisi burada taklit edilmez. Tohum veriler yalnizca stock LiderAhenk'in bekledigi ortami dogru kurmak icin vardir.
