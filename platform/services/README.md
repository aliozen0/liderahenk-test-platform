# Platform Services

This directory contains platform-owned sidecar services.

- Use this path for services that belong to the external platform layer rather
  than the upstream LiderAhenk product surface.
- These services usually provide orchestration, evidence, or platform-only
  support capabilities.

Current example:

- `registration-orchestrator/`

If the service is part of the composed runtime product surface, use
`/home/huma/liderahenk-test/services/`.
