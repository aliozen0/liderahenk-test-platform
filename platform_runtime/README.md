# Platform Runtime

This package contains executable runtime truth and reporting code.

- Readiness checks, registration evidence, scenario support summaries, baseline
  validation, and runtime reporting live here.
- `platform/` holds declarative contracts and manifests.
- `platform_runtime/` executes those contracts against a running system.

If you are defining a new contract or manifest, start in
`/home/huma/liderahenk-test/platform/`.
If you are implementing runtime verification logic, start here.
