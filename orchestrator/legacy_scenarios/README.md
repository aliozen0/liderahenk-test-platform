# Legacy Scenarios

This directory contains scenario files for the legacy/simple orchestrator CLI.

- Files here are consumed by `orchestrator/cli.py`.
- `make run-legacy-scenario S=<name>.yml` resolves scenarios from this directory.
- These scenarios are not the canonical release-gate scenario packs.

If you need the active scenario-pack surface used by runtime operational checks
and acceptance reporting, use
